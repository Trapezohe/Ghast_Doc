# Settings Reference

## Current category map

| Category | Primary content | Depends on Companion? |
| --- | --- | --- |
| Wallet | Create, import, unlock, export, and automatic locking | No |
| Remote | Telegram remote control and remote status | No |
| Companion | Connection, local policy, workspace, diagnostics, and repair | Yes |
| AI Models | Model enablement, balances, recharge, refund, and statistics | No |
| Security | Security settings and restrictions | Partially |
| Language | Language switching | No |
| Appearance | Visual theme settings | No |
| Env Vars | Environment variables | Partially |
| Scheduled Tasks | Scheduled tasks and heartbeat | Partially |

## Scope boundary

- The presence of a category does not mean every control under it is available in every environment.
- Companion-related categories should always distinguish between "the service is connected" and "a configuration entry exists."
