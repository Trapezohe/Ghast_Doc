# 记忆文件参考

## 当前核心文件映射

| 内部规范名 | 当前 UI 标签 | 当前用途 |
| --- | --- | --- |
| `soul` | `soul.md` | 代理自身设定 |
| `user` | `profile.md` | 用户画像 / 偏好 |
| `strategy` | `strategy.md` | 行动与策略规则 |
| `memory` | `long-term-memory.md` | 长期记忆主体 |
| `heartbeat` | `heartbeat.md` | 心跳检查清单 |

## 当前边界

- 工具层会把这些文件视为保留核心记忆，不允许误写成普通工作区文件。
- 文档里同时保留“内部规范名”和“当前 UI 标签”，是为了避免排障时对不上号。
