# 设置项参考

## 当前分类参考

| 分类 | 主要内容 | 是否依赖 Companion |
| --- | --- | --- |
| Wallet | 创建 / 导入 / 解锁 / 导出 / 自动锁定 | 否 |
| Remote | Telegram 远控与远程状态 | 否 |
| Companion | 连接、本地策略、工作区、诊断、修复 | 是 |
| AI Models | 模型启用、余额、充值、退款、统计 | 否 |
| Security | 安全设置与限制 | 部分依赖 |
| Language | 语言切换 | 否 |
| Appearance | 外观主题 | 否 |
| Env Vars | 环境变量 | 部分依赖 |
| Scheduled Tasks | 定时任务与 Heartbeat | 部分依赖 |

## 写文档时的边界

- 分类存在，不代表该分类下所有按钮在任何环境下都可用。
- Companion 相关分类应始终把“服务已连接”和“只是有配置入口”区分开。
