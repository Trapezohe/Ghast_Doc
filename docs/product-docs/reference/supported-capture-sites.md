# 支持的采集站点

## 当前内容脚本匹配列表

| 类别 | 当前站点 |
| --- | --- |
| 0G | `*.0g.ai`、`chainscan.0g.ai` |
| 区块浏览器 | `*.etherscan.io`、`*.bscscan.com`、`*.polygonscan.com`、`*.arbiscan.io` |
| DeFi | `app.uniswap.org`、`*.aave.com`、`*.lido.fi`、`*.sushi.com`、`*.compound.finance`、`*.curve.fi`、`*.pancakeswap.finance` |
| 社交 | `x.com`、`twitter.com` |

## 当前模式说明

- `web3`：只在这些站点及其识别页面类型上返回上下文。
- `off`：不返回页面上下文。
