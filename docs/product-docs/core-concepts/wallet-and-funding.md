# 钱包与模型充值

## 本页说明

本页帮助你从整体上理解 Ghast AI 中的钱包与模型资金关系，以及普通用户在首次使用时应按什么顺序进入这条资金路径。

## 先分清两层资金

| 层级 | 对普通用户意味着什么 |
| --- | --- |
| 钱包 | 你的本地资金入口 |
| 模型资金 | 为某个模型准备的可用额度 |

先分清这两层，后面的充值、补充和退款就不会那么容易混淆。

## 标准使用顺序

更稳妥的顺序通常是：

1. 先通过钱包建立本地资金入口。
2. 再在模型页面里选择一个主模型。
3. 再为这个模型准备起步资金。
4. 真正使用后，再根据消耗情况继续追加或调整。

这比一开始就给很多模型同时充值，更适合作为日常使用方式。

## 什么时候不需要急着充值

如果你还只是熟悉产品、浏览页面或完成基础设置，通常不需要急着给多个模型准备资金。

更推荐的做法是：先选一个主要模型，先给它准备足够开始使用的资金，真正使用后，再决定是否继续补充。

## 退款该怎么理解

退款更适合被理解成“把当前可退部分收回”，而不是“所有可见金额都能立刻全部退回”。

对普通用户来说，最重要的是理解：

- 可退额度和当前可退范围有关。
- 可退额度不一定等于页面上看到的全部余额。
- 刚充值不久的部分，可能不会立刻进入可退范围。

因此，看到差异时，优先把它理解为当前阶段不同，而不是直接判断异常。

Ghast AI 当前将钱包与模型资金分成两层处理：钱包用于建立本地资金入口，模型资金用于支持实际模型使用。对普通用户而言，更稳妥的使用方式是先完成钱包设置，再围绕一个主要模型逐步准备和管理可用资金。
